# Daily Threat Intel Digest - April 3, 2026

## 🔴 Critical Threats & Active Exploitation

**[UPDATE] European Commission Cloud Breach Scope Expands to 30 EU Entities**
Following last week's ShinyHunters claim of breaching the European Commission’s Amazon cloud environment, CERT-EU has confirmed the real-world impact: the stolen 90GB dataset contains personal data, usernames, and email content affecting not just the Commission’s internal staff, but at least 29 other Union entities using the europa.eu web hosting service. The threat actors leveraged a compromised AWS management API key—stolen during the wider Trivy supply-chain attack—to access the cloud environment, used TruffleHog to search for additional secrets, and created new access keys to evade detection before exfiltrating data. While no websites were defaced or taken offline, the exposure of over 51,000 files containing outbound email communications—including automated "bounce-back" notifications that capture original user-submitted content—creates a substantial intelligence goldmine for state-sponsored phishing and espionage operations targeting European officials [[BleepingComputer](https://www.bleepingcomputer.com/news/security/cert-eu-european-commission-hack-exposes-data-of-30-eu-entities/)]. 

**[NEW] Qilin Ransomware Deploys Custom Kernel Drivers to Blind 300+ EDR Solutions**
The Qilin ransomware group has operationalized a sophisticated, multi-stage infection chain that systematically dismantles endpoint security before deploying its encryption payload. Analyzed by Cisco Talos, the attack leverages a malicious DLL (`msimg32.dll`) to execute entirely in memory, using structured exception handling (SEH) and direct syscall scanning to bypass user-mode behavioral monitoring. The malware then loads two kernel-level drivers—a legitimate, signed `rwdrv.sys` renamed as "ThrottleStop.sys" for direct physical memory access, and a custom `hlpdrv.sys` to terminate protected EDR processes. By iterating through a hardcoded list of over 300 EDR drivers, the malware systematically unregisters monitoring callbacks for process creation, thread creation, and image loading. It temporarily disables Windows Code Integrity enforcement by overwriting kernel validation functions, blinding security tools at the root level before restoring the integrity checks to minimize forensic evidence [[Cyberpress](https://cyberpress.org/qilin-ransomware-4/); [GBHackers](https://gbhackers.com/qilin-ransomware-deploys-malicious-dll/)]. 

**[NEW] Iranian Handala Group Breaches Israeli Defense Contractor, Exposes IDF Schematics**
The Iranian state-sponsored threat actor Handala (aka Void Manticore) has claimed responsibility for a significant breach of PSK Wind Technologies, a primary architect of Israel’s military command centers and air defense communications. The threat actors published internal documents, photographs of active IDF command centers, and, most critically, deep technical schematics of advanced communications hardware. Handala claims the intelligence has been transferred to missile units within the Axis of Resistance, elevating this intrusion beyond cyber espionage into a potential enabler for kinetic operations. The attack mirrors their March 2026 enterprise-scale wiper attacks, demonstrating a continued pattern of exploiting trusted third-party vendors to bypass hardened military network perimeters [[Cyberpress](https://cyberpress.org/iranian-hacker-group-handala-claims-breach-of-israeli-defense-contractor/)].

## 🎯 Threat Actor Activity & Campaigns

**[NEW] North Korean Threat Actors Pivot to GitHub for C2 in LNK Phishing Campaigns**
A newly uncovered North Korean-linked phishing campaign targeting organizations in South Korea is abusing the GitHub API as a stealthy Command and Control (C2) server. Discovered by FortiGuard Labs, the attack uses heavily obfuscated malicious LNK (shortcut) files to silently drop a decoy PDF while executing a hidden PowerShell script. The script establishes a "keep-alive" connection to private GitHub repositories using hardcoded access tokens, allowing the malware to continuously gather network configuration details and fetch additional malicious modules. By using the `motoralis` GitHub account as an operational hub, the threat actors ensure their data exfiltration and payload hosting remain hidden in plain sight, riding on the high reputation of the GitHub domain to bypass standard network defenses [[Cyberpress](https://cyberpress.org/github-c2-in-lnk-phishing/); [GBHackers](https://gbhackers.com/lnk-phishing-campaign/)].

**[NEW] Threat Actors Spoof CERT-UA to Deploy Go-Based AGEWHEEZE RAT**
Between March 26 and 27, threat actors operating under the identifier UAC-0255 (and self-identifying as "CYBER SICKLE") aggressively impersonated Ukraine's own Computer Emergency Response Team (CERT-UA) in a phishing campaign. The attackers sent deceptive emails urging recipients to download a "specialized protection software" archive hosted on Files.fm. The deployed malware, a Go-based Remote Access Trojan tracked as AGEWHEEZE, grants attackers complete remote control, supporting live screen broadcasting, remote mouse/keyboard emulation, and clipboard reading. Using WebSockets for C2 communication, the malware establishes deep persistence by modifying the registry and creating hidden scheduled tasks like `SvcHelper` and `CoreService` to elevate privileges [[Cyberpress](https://cyberpress.org/cert-ua-spoofing-drops-gorat/)].

**[UPDATE] Axios npm Supply Chain Attack Attributed to STARDUST CHOLLIMA**
CrowdStrike has attributed the March 31 Axios npm compromise to the North Korean group STARDUST CHOLLIMA (with moderate confidence), noting a major evolution in the deployed ZshBucket malware. Previously limited to macOS, the newly discovered ZshBucket variants now actively target Linux and Windows systems, scanning infected OS environments to quietly gather user and host data. The underlying motive is assessed as financial, consistent with STARDUST CHOLLIMA's aggressive focus on generating illicit currency—frequently targeting cryptocurrency and fintech sectors through massive supply chain dragnets [[Cyberpress](https://cyberpress.org/axios-npm-package-compromised/); [GBHackers](https://gbhackers.com/axios-npm-in-supply-chain/)].

**[NEW] Babuk-Based Ransomware Impersonates Akira to Extort South American Organizations**
ESET researchers have uncovered a deceptive ransomware campaign actively targeting Windows systems in South America using a modified version of the leaked Babuk ransomware builder. The unidentified threat actors deliberately leave behind visual hallmarks of the Akira ransomware group—including familiar ransom notes and file extensions—to confuse incident responders and psychologically pressure victims into paying quickly. This tactic highlights a growing trend of false-flag operations in the cybercriminal underworld, requiring security teams to rely on deep malware analysis rather than surface-level incident indicators for accurate attribution [[Cyberpress](https://cyberpress.org/akira-like-ransomware-hits-south-america/)].

## ⚠️ Vulnerabilities & Patches

**[NEW] Critical OpenSSH 10.3 Patches Dangerous Shell Injection Flaw**
The OpenSSH project has released version 10.3 to patch multiple security vulnerabilities, most notably a dangerous shell injection flaw in the SSH client. Attackers could execute arbitrary shell commands by crafting malicious usernames passed via the command line when configuration files used specific tokens like `%u`. The update also fixes a flaw allowing certificates with comma-separated names to bypass `authorized_keys` restrictions, a legacy SCP bug that failed to clear `setuid/setgid` permissions when downloading files as root, and an issue where restricting an ECDSA key inadvertently permitted any other ECDSA algorithm. Administrators are urged to upgrade immediately, as OpenSSH 10.3 also introduces stricter validation rules, drops support for older cryptographic rekeying implementations, and strengthens anti-spam penalties against bot scanning [[Cyberpress](https://cyberpress.org/openssh-10-3-fixes/); [GBHackers](https://gbhackers.com/openssh-10-3-released-with-patch-for-shell-injection/); [Canadian Centre for Cyber Security](https://cyber.gc.ca/en/alerts-advisories/openssh-security-advisory-av26-312)].

**[NEW] Perfmatters WordPress Plugin RCE Vulnerability Affects 200,000 Sites**
A critical vulnerability (CVE-2026-4350) in the widely used Perfmatters WordPress plugin allows unauthenticated threat actors to delete arbitrary files, ultimately enabling complete site takeover. Due to missing capability checks and unsanitized `$_GET['delete']` parameters in the `action_handler()` function, attackers can use path traversal sequences to delete the core `wp-config.php` file. Deleting this configuration file forces WordPress into its installation wizard, allowing the attacker to connect the site to a database under their control. Site administrators running version 2.5.9.1 or earlier must immediately update to the patched version 2.6.0 [[Wordfence](https://www.wordfence.com/blog/2026/04/200000-wordpress-sites-affected-by-arbitrary-file-deletion-vulnerability-in-perfmatters-wordpress-plugin/)].

**[NEW] Unauthenticated Arbitrary File Deletion in TP-Link Tapo Security Cameras**
TP-Link has addressed a batch of severe vulnerabilities in its Tapo C520WS security camera system that allowed attackers to launch Denial-of-Service (DoS) attacks, crash the devices, and alter settings without authentication. Because security cameras are critical physical security controls, knocking these devices offline or manipulating their configurations creates direct physical security risks to homes and businesses [[GBHackers](https://gbhackers.com/tp-link-router-flaws-allowed-attackers-to-launch-dos-attacks/)].

## 📋 Policy & Industry News

**[NEW] ICE Confirms Use of Paragon Spyware for Fentanyl Investigations**
Immigration and Customs Enforcement (ICE) has officially confirmed its use of Paragon's commercial spyware, previously found on the phones of journalists and suspected of use by multiple foreign governments. Acting ICE Director Todd Lyons stated the technology was authorized to help Homeland Security Investigations fight fentanyl trafficking via encrypted communications, asserting compliance with a 2023 executive order requiring certification that commercial spyware doesn't pose national security risks. The disclosure has prompted immediate outrage from a trio of House Democrats, who criticized the agency for deploying the surveillance tool with "no Congressional oversight and a complete lack of compliance standards" [[CyberScoop](https://cyberscoop.com/ice-using-paragon-spyware-house-democrats-letter/)].

**[NEW] Malicious GitHub Repos Exploit Claude Code Leak to Push Vidar Stealer**
Threat actors are capitalizing on the recent accidental leak of Anthropic's Claude Code source code by SEO-optimizing malicious GitHub repositories to appear at the top of Google Search results. Users searching for the leak are tricked into downloading a 7-Zip archive containing a Rust-based executable, which deploys the Vidar infostealer and the GhostSocks proxying tool. This campaign demonstrates how quickly high-profile source code leaks can be weaponized to distribute commodity malware to curious developers and researchers looking for unrestricted enterprise features [[BleepingComputer](https://www.bleepingcomputer.com/news/security/claude-code-leak-used-to-push-infostealer-malware-on-github/)].

**[UPDATE] Microsoft Begins Forced Upgrades to Windows 11 25H2**
Microsoft has commenced automatic, forced upgrades for unmanaged devices running Windows 11 24H2 Home and Pro editions to Windows 11 25H2, six months ahead of the older version's October 13, 2026 end-of-support date. IT administrators managing enterprise environments should be aware of this aggressive lifecycle shift to avoid user disruption, particularly as the rollout includes emergency updates issued since March Patch Tuesday to resolve critical issues breaking Microsoft account sign-ins and RRAS remote code execution flaws [[BleepingComputer](https://www.bleepingcomputer.com/news/microsoft/microsoft-now-force-upgrades-unmanaged-windows-11-24h2-pcs/)].
