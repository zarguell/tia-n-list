# Daily Threat Intel Digest - April 8, 2026

## 🔴 Critical Threats & Active Exploitation

**[UPDATE] Russian GRU (APT28) router hijacking campaign disrupted by FBI**  
The FBI has successfully disrupted "Operation Masquerade," a sprawling DNS hijacking campaign conducted by Russian GRU Military Unit 26165 (APT28/Forest Blizzard) that compromised over 18,000 SOHO routers across 120+ countries. The operation, executed via court-authorized remote access, reset malicious DNS configurations and blocked attacker access channels within the U.S. The campaign, active since August 2025, exploited vulnerabilities primarily in TP-Link routers to hijack traffic, allowing attackers to intercept credentials and OAuth tokens via AiTM attacks spoofing services like Microsoft Outlook Web Access. While U.S. remediation is underway, global devices remain vulnerable; organizations must verify DNS settings, replace outdated SOHO routers, and enforce Zero Trust DNS policies to prevent edge devices from being weaponized for espionage [[CyberScoop](https://cyberscoop.com/forest-blizzard-apt28-routers-espionage-campaign-operation-masquerade/); [FBI/DOJ](https://www.justice.gov/opa/pr/justice-department-conducts-court-authorized-disruption-dns-hijacking-network-controlled); [Microsoft](https://www.microsoft.com/en-us/security/blog/2026/04/07/soho-router-compromise-leads-to-dns-hijacking-and-adversary-in-the-middle-attacks/); [GBHackers](https://gbhackers.com/fbi-takes-down-russian-campaign/); [CyberPress](https://cyberpress.org/fbi-disrupts-russian-router-hijacking/)].

**[UPDATE] UNC1069 (BlueNoroff) weaponizes fake meeting lures for browser-based attacks**  
North Korean threat actor UNC1069 is bypassing traditional malware installation requirements by tricking cryptocurrency and open-source software targets into executing malicious scripts via fake Zoom and Microsoft Teams browser prompts. Over the past two months, SEAL blocked 164 domains tied to this campaign. Attackers spend weeks building trust on platforms like LinkedIn and Slack before scheduling meetings via legitimate Calendly links. Instead of downloading executables, victims are lured into running small scripts or pasting commands, triggering malware that steals browser passwords, cryptocurrency wallets, and session tokens—enabling subsequent supply chain compromises like the recent Axios npm package hijack [[CyberPress](https://cyberpress.org/fake-meeting-lures-weaponized/); [GBHackers](https://gbhackers.com/fake-zoom-teams-calls/)].

**[NEW] Ninja Forms WordPress plugin actively exploited for unauthenticated RCE**  
Attackers are actively exploiting CVE-2026-0740 (CVSS 9.8), a critical arbitrary file upload vulnerability in the Ninja Forms File Upload premium add-on for WordPress, to achieve remote code execution. The flaw allows unauthenticated attackers to upload malicious PHP scripts and manipulate filenames for path traversal due to missing file type validation. Wordfence has blocked over 3,600 attacks in the past 24 hours targeting the 90,000-customer install base. A patch is available in version 3.3.27; immediate upgrades are essential to prevent web shell deployment and complete site takeovers [[BleepingComputer](https://www.bleepingcomputer.com/news/security/hackers-exploit-critical-flaw-in-ninja-forms-wordpress-plugin/)].

## 🎯 Threat Actor Activity & Campaigns  

**[NEW] Masjesu botnet operates stealthy DDoS-for-hire service targeting IoT**  
The Masjesu botnet is emerging as a sophisticated commercial DDoS-for-hire operation, targeting routers and gateways across ARM, MIPS, SPARC, and AMD64 architectures. The malware maintains stealth by actively scanning and avoiding high-profile blocklisted networks like U.S. DoD ranges, utilizing XOR-based encryption to hide C2 domains, and masquerading as the 32-bit Linux dynamic linker. Masjesu propagates by exploiting known flaws in D-Link, Netgear, Huawei, and GPON devices to download recruitment shell scripts. Network admins should monitor for the unique "masjesu" user-agent string and unexpected cron jobs executing every 15 minutes, while ensuring default credentials on IoT devices are changed immediately [[CyberPress](https://cyberpress.org/masjesu-botnet-targets-routers/)].

**[NEW] ComfyUI AI servers hijacked for fileless cryptomining and proxy botnets**  
Threat actors are aggressively exploiting internet-exposed ComfyUI servers—AI image generation platforms running on high-end cloud GPUs—to deploy fileless cryptominers and stealthy proxy botnets. Attackers abuse ComfyUI’s custom node ecosystem, which inadvertently accepts raw Python code, functioning as an unauthenticated RCE endpoint. The secondary "Ghost" payload deploys XMRig and lolMiner directly into memory using Linux memfd calls, leaving no disk artifacts. The malware also actively hunts rival cryptomining operations, including a competing botnet named Hisana, redirecting their mining output to the attacker's wallet. Over 1,000 ComfyUI instances are currently exposed, with 214 confirmed vulnerable; organizations running AI workloads must ensure these interfaces are not publicly accessible [[CyberPress](https://cyberpress.org/comfyui-servers-hijacked-globally/); [GBHackers](https://gbhackers.com/comfyui-servers-hijacked/)].

**[NEW] Remus Infostealer debuts as Lumma Stealer successor**  
A new 64-bit infostealer dubbed Remus has surfaced, bearing strong code similarities to the notorious Lumma Stealer. Its emergence follows the 2025 law-enforcement disruption and public doxxing of Lumma's core operators. Remus mirrors Lumma's core capabilities—harvesting browser passwords, cookies, and autofill data—indicating a rapid rebranding or operational pivot by the original developers to rebuild their theft infrastructure under a new name [[GBHackers](https://gbhackers.com/remus-infostealer-debuts/)].

## ⚠️ Vulnerabilities & Patches

**[NEW] Claude AI discovers 13-year-old RCE in Apache ActiveMQ**  
Anthropic’s Claude AI model uncovered CVE-2026-34197, a critical remote code execution vulnerability in Apache ActiveMQ Classic that went undetected for 13 years. The flaw exists in the Jolokia REST API, where an attacker can abuse the `addNetworkConnector` operation to supply a malicious `vm://` URI, forcing the broker to download and execute a remote Spring XML configuration file. While the API requires admin credentials, versions 6.0.0 through 6.1.1 are also affected by CVE-2024-32114, which inadvertently removed authentication, creating an unauthenticated RCE chain. Organizations must upgrade to ActiveMQ Classic 5.19.4 or 6.2.3, change default credentials, and monitor logs for `vm://` URIs and suspicious POST requests to `/api/jolokia/` [[CyberPress](https://cyberpress.org/claude-discovers-13-year-old-rce-vulnerability-in-apache-activemq-within-minutes/); [GBHackers](https://gbhackers.com/claude-identifies-critical-13-year-old-rce-vulnerability-in-apache-activemq/)].

**[NEW] Claude Code finds auth bypass in NSA's Ghidra Server**  
AI-assisted auditing discovered a critical authentication bypass in NSA's Ghidra Server PKI mode. A flaw in `PKIAuthenticationModule.java` allows execution to skip signature verification entirely when `sigBytes` is null, falling through to resolve the certificate's subject DN as an authenticated identity without requiring proof of private key possession. This enables low-privileged analysts to escalate to administrator, exfiltrate shared reverse engineering databases, and rewrite repository access controls. A PoC and fix are available on GitHub [[Malware.News](https://malware.news/t/mad-bugs-claude-found-an-auth-bypass-in-nsas-ghidra-server/105838#post_1)].

**[NEW] Kubernetes misconfigurations fuel 282% surge in cloud account breaches**  
Over the past year, theft of Kubernetes service account tokens has surged by 282%, with 78% of targeted attacks hitting the IT sector. Attackers are pivoting from complex container escapes to simply exploiting exposed applications and overprivileged identities. Notable incidents include North Korean Lazarus group breaching a major cryptocurrency exchange by deploying a malicious pod to extract a highly privileged service account token, bypassing perimeter security to access backend financial systems. Security teams must enforce strict RBAC, replace permanent credentials with short-lived tokens, and implement deep runtime monitoring to detect anomalous container behavior like unexpected hacking script downloads [[CyberPress](https://cyberpress.org/kubernetes-flaws-expose-clouds/)].

## 🛡️ Defense & Detection

**[NEW] Fiber optic cables can be converted into covert listening devices**  
Researchers demonstrated that standard telecom fiber optic cables can be turned into hidden microphones capable of capturing conversations from up to two meters away. By connecting a Distributed Acoustic Sensing (DAS) system to a Fiber-to-the-Home (FTTH) connection, attackers can detect microscopic cable deformations caused by sound waves and reconstruct over 80% of spoken content using AI speech recognition. Coiling 15 meters of fiber inside a standard wall-mounted internet box increases sensitivity. The attack operates from 50 meters away through the optical network, emits no radio signals, and evades standard RF scanners and ultrasonic jammers. Physical security teams in sensitive environments should minimize excess indoor fiber loops and consider optical isolators to prevent signal leakage [[CyberPress](https://cyberpress.org/fiber-optic-cables-can-be-turned-into-hidden-microphones-to-spy-on-conversations/); [GBHackers](https://gbhackers.com/fiber-optic-cables-spy-on-private-conversations/)].

**[NEW] Claude Code leak weaponized to deliver Vidar and GhostSocks**  
Threat actors are actively poisoning GitHub Releases with Vidar infostealer and GhostSocks proxy malware, disguising the payloads as "leaked" Anthropic Claude Code source code. This campaign highlights how governance failures around AI development tools are rapidly cascading into traditional compromise vectors, exploiting developer curiosity and the rush to adopt AI tooling [[GBHackers](https://gbhackers.com/claude-code-leak/)].
