# Daily Threat Intel Digest - April 21, 2026

## 🔴 Critical Threats & Active Exploitation

**[NEW] North Korean Lazarus Group deploys new macOS malware kit in business compromise campaign**
Lazarus Group is actively targeting business professionals through Telegram-based social engineering that tricks users into executing terminal commands to install the "Mach-O Man" malware kit, a multi-stage macOS threat capable of stealing credentials, browser sessions, and macOS Keychain data. The campaign uses fake meeting invitations mimicking Zoom, Teams, and Google Meet platforms, with compromised contacts sending malicious links. The malware establishes persistence via LaunchAgents and exfiltrates data through Telegram's API, blending malicious traffic with legitimate communications. Analysis reveals poorly written code with infinite loop bugs, but the threat remains significant for enterprises with macOS developer workstations that often access source code repositories and cloud credentials [[ANY.RUN](https://any.run/cybersecurity-blog/lazarus-macos-malware-mach-o-man/); [BleepingComputer](https://www.bleepingcomputer.com/news/security/kelpdao-suffers-290-million-heist-tied-to-lazarus-hackers/)].

**[NEW] Malicious GGUF models achieve RCE on SGLang inference servers via template injection**
Attackers can achieve remote code execution on SGLang inference servers by crafting malicious GGUF model files that exploit a Server-Side Template Injection (SSTI) vulnerability in the framework's reranking functionality (CVE-2026-5760). The flaw stems from SGLang using Jinja2's default insecure Environment() function instead of ImmutableSandboxedEnvironment to process chat templates embedded in model metadata, allowing attackers who control model files on platforms like Hugging Face to execute arbitrary Python code. This represents a growing AI supply chain risk where untrusted model files can compromise entire inference infrastructure, similar to previous "Llama Drama" vulnerabilities in llama-cpp-python and vLLM frameworks [[Cyberpress](https://cyberpress.org/hackers-could-weaponize-gguf-models-to-achieve-rce-on-sglang-inference-servers/); [GBHackers](https://gbhackers.com/malicious-gguf-models-could-trigger-rce/)].

**[NEW] Axios npm package supply chain attack distributed RAT to thousands of developers**
CISA has issued an urgent alert regarding compromise of the widely-used Axios JavaScript HTTP library, where attackers injected malicious code into versions 1.14.1 and 0.30.4 on March 31, 2026. The compromised versions installed a hidden dependency called plain-crypto-js 4.2.1 that functions as a remote access trojan, connecting to attacker-controlled infrastructure at Sfrclak[.]com to download additional payloads. Organizations running affected versions must immediately downgrade to Axios 1.14.0 or 0.30.3, remove node_modules/plain-crypto-js/ directories, and rotate all potentially exposed credentials including cloud access keys, npm tokens, SSH keys, and CI/CD secrets. The breach highlights ongoing risks in software supply chains where trusted package ecosystems become attack vectors for lateral movement into development environments and production systems [[Cyberpress](https://cyberpress.org/cisa-warns-axios-npm-package-was-compromised-in-major-supply-chain-attack/); [GBHackers](https://gbhackers.com/cisa-warns-compromised-axios-npm-package/)].

## 🎯 Threat Actor Activity & Campaigns

**[NEW] UNC1069 (North Korea) targets crypto professionals with fake VC personas**
A financially motivated North Korean threat actor overlapping with the Bluenoroff cluster is conducting sophisticated social engineering campaigns against cryptocurrency and Web3 professionals using fabricated venture capital identities like "WallEye Capital," "Web3BitCapital," and "SolidBit Capital." The operation leverages compromised Telegram and LinkedIn accounts to build rapport before sending Calendly scheduling links to fraudulent meeting environments that mimic Google Meet, Zoom, and Microsoft Teams. Cross-platform malware delivered via ClickFix techniques includes Cabbage RAT variants for Windows and Linux, and NukeSped RAT for macOS, with capabilities to steal cryptocurrency wallet data and establish persistent access. Attribution to North Korea suggests the campaign funds regime military and espionage operations [[Cyberpress](https://cyberpress.org/unc1069-targets-crypto-pros/)].

**[NEW] SideWinder APT harvests government webmail credentials via Zimbra phishing clone**
The SideWinder advanced persistent threat group is conducting targeted credential harvesting against South Asian government and military entities including the Bangladesh Navy and Pakistan Ministry of Foreign Affairs using a pixel-perfect Zimbra webmail clone hosted on Cloudflare Workers. The phishing kit displays authentic stolen diplomatic documents as blurred lures that create urgency before redirecting to fake login pages that dynamically load CSS and assets directly from legitimate organizational servers to maximize authenticity. A critical programming error by the kit developer exposed their Linux home directory through an unhandled exception, providing defenders with exact signatures to track the actor's infrastructure [[Cyberpress](https://cyberpress.org/sidewinder-hits-gov-webmail/); [GBHackers](https://gbhackers.com/fake-chrome-pdf-viewer/)].

**[NEW] Former ransomware negotiator pleads guilty to conspiring with BlackCat**
Angelo Martino, a 41-year-old former employee of cybersecurity incident response company DigitalMint, has pleaded guilty to conspiring with BlackCat ransomware operators to extort U.S. victims, marking the third conviction in this insider threat case. Martino shared confidential victim negotiation positions and insurance policy limits with BlackCat affiliates while working as a legitimate ransomware negotiator, enabling the threat actors to maximize extortion demands. The conspiracy involved at least five U.S. organizations including a financial services firm that paid $25.6 million and a nonprofit that paid $26.8 million. Martino's accomplices Ryan Goldberg and Kevin Martin previously pleaded guilty in December and face sentencing on April 30 [[BleepingComputer](https://www.bleepingcomputer.com/news/security/former-ransomware-negotiator-pleads-guilty-to-blackcat-attacks/); [Malware News](https://malware.news/t/florida-man-working-as-a-ransomware-negotiator-pleads-guilty-to-conspiracy-to-deploy-ransomware-and-extort-u-s-victims/106272#post_1)].

## 🛡️ Defense & Detection

**[NEW] macOS living-off-the-land techniques enable stealthy lateral movement**
Security researchers have documented extensive native macOS primitives that adversaries can weaponize for lateral movement and execution while evading traditional SSH telemetry monitoring. Techniques include Remote Application Scripting (RAS) using the eppc:// protocol to execute commands through Terminal.app as a proxy, Spotlight metadata abuse via Finder comments to stage payloads in file metadata rather than data forks, and native protocols like SMB, Netcat, Git, TFTP, and SNMP for tool transfer outside SSH visibility. Defenders should monitor for process lineage patterns like launchd → AppleEventsD → Terminal → bash, frequent mdls calls to Spotlight metadata, and inbound TCP traffic on port 3031 (eppc). MDM policies should disable RAS and unnecessary network services by default [[Talos Intelligence](https://blog.talosintelligence.com/bad-apples-weaponizing-native-macos-primitives-for-movement-and-execution/); [Malware News](https://malware.news/t/bad-apples-weaponizing-native-macos-primitives-for-movement-and-execution/106270#post_1)].

**[NEW] Fake TikTok downloader extensions steal data from 130,000 browser users**
A coordinated campaign involving at least 12 malicious browser extensions disguised as TikTok video downloaders has compromised over 130,000 Chrome and Edge users, with approximately 12,500 installations still active. The extensions use Manifest V3 to retrieve remote configuration after installation, enabling dynamic activation of tracking capabilities that collect browsing patterns, download metadata, system language, timezone, and battery status to build unique device fingerprints. Several extensions achieved "Featured" status on official stores, increasing download rates and user trust. The campaign demonstrates how extension validation focuses on initial submission while allowing post-installation behavior changes that bypass traditional defenses [[Cyberpress](https://cyberpress.org/fake-tiktok-downloader-extensions/); [GBHackers](https://gbhackers.com/12-fraudulent-browser-extensions-disguised-as-tiktok-downloaders/)].

**[NEW] AI models accelerate vulnerability discovery, shrinking defender patch windows**
Unit 42 testing reveals frontier AI models can autonomously discover zero-day vulnerabilities and chain complex exploitation paths, particularly effective against open-source software where source code transparency allows threat actors to rigorously test for weaknesses without network defender visibility. The most immediate threat is collapse of traditional patching windows—AI-assisted attackers can exploit vulnerabilities within hours of discovery rather than days, outpacing manual monitoring and remediation processes. Frontier models demonstrate only marginal improvement over existing AI tools when analyzing compiled executables, but show exceptional capability identifying deep vulnerabilities in raw source code. Organizations should implement continuous monitoring strategies that detect abnormal behavior like suspicious network requests and unauthorized permission changes rather than relying solely on patching cadence [[Cyberpress](https://cyberpress.org/ai-threatens-patch-window/); [GBHackers](https://gbhackers.com/ai-driven-exploitation/)].
